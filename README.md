# Weather Prediction Project

This project presents a Bayesian model for predicting weather conditions based on historical hourly
weather data.


![Python](https://img.shields.io/badge/Python-3.x-blue)
![Status](https://img.shields.io/badge/Project-Completed-green)
## Project Files
- `project.ipynb` – Main Jupyter Notebook containing analysis and predictions.
- `weather.csv` – Dataset used for analysis.

## How to Use
1. Make sure you have **Python 3.x** installed.
2. Install the required libraries if you haven’t already:
   ```bash
   pip install pandas matplotlib seaborn scikit-learn

Open project.ipynb in Jupyter Notebook, JupyterLab, or VS Code.

Run the notebook cells in order to reproduce the analysis and visualizations.

### Overview

- Loads historical weather data from weather.csv.

- Performs data cleaning, visualization, and analysis.

- Applies basic prediction techniques to forecast weather trends.

### Notes

- Designed as a beginner-friendly example of data analysis and prediction.

- You can expand the project by adding machine learning models or more advanced forecasting methods.
